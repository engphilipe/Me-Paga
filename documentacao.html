Me Paga - Docs Acessar Sistema

Guia de Integração

Aprenda como conectar o Me Paga ao seu fluxo de atendimento no WhatsApp para gerar cobranças PIX dinâmicas e automatizadas.

Visão Geral da Arquitetura

Nosso sistema funciona de forma assíncrona. Seu Chatbot manda uma requisição com o valor para a nossa API, nós retornamos o QR Code. Quando o cliente paga, o Gateway avisa o nosso sistema via Webhook, e nós notificamos seu sistema do recebimento.

Passo a Passo

1

Gerar Chave de API

Acesse o Dashboard > Configurações e clique em "Gerar Nova Chave API". Utilize esta chave no Header de todas as requisições HTTP para a nossa API.

2

Criar a Cobrança via Chatbot

Quando o cliente fechar a compra no WhatsApp, faça um POST para nosso endpoint enviando os dados do pedido.

POST https://api.mepaga.com.br/v1/cobrancas Authorization: Bearer SUA_CHAVE_API Content-Type: application/json { "cliente": "João Silva", "valor": 150.00, "referencia": "pedido_102" }

A API responderá com a imagem do QR Code e o código Pix Copia e Cola para você enviar ao cliente.

3

Configurar Webhook de Confirmação

Vá em Configurações e cadastre a URL de Webhook do seu Chatbot. Sempre que um PIX for pago com sucesso, nós enviaremos um `POST` para a sua URL avisando.

POST https://seu-chatbot.com/webhook/mepaga { "id_transacao": "1a2b3c4d", "referencia": "pedido_102", "status": "pago", "valor_pago": 150.00 }
⚠️ Atenção à Segurança: Valide se a requisição que chega no seu Webhook realmente veio do IP do Me Paga ou valide o payload com a chave de assinatura disponível no seu Dashboard.
